I'm excited to launch Roadmap Skill - a visual project management tool that brings Kanban boards to your AI workflow.

**Why I built this:**
I love using AI assistants like Claude and Cursor, but managing projects through text-only interfaces felt limiting. I wanted the best of both worlds - AI-powered task creation and organization, plus a visual Kanban board when I need to see the big picture.

**How it works:**
1. Run `npx roadmap-skill-web` to launch the web UI
2. Or integrate with Claude/Cursor via MCP protocol
3. Ask your AI to create projects and tasks
4. Drag and drop tasks to update status
5. Track progress visually

**Tech stack:**
- MCP (Model Context Protocol) for AI integration
- React + TypeScript for the web UI
- Express backend
- Published on npm
